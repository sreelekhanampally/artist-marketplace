# 🎨 Artist Marketplace

A full-stack marketplace that enables artists to build professional portfolios while showcasing and selling their artwork online. Customers can explore artwork collections, browse artist profiles, manage their shopping cart, and complete secure purchases through Razorpay.

---

## ✨ Features

### 👨‍🎨 Artist

- Create and manage artist profiles
- Showcase artwork collections
- Upload and organize artworks
- Manage customer orders

### 🛍️ Customer

- User registration & authentication
- Browse artist portfolios
- Explore artwork collections
- View artwork details
- Shopping cart management
- Secure checkout
- Order history

### 🔐 Authentication

- JWT Authentication
- Password hashing with bcrypt
- Protected routes

### 💳 Payments

- Razorpay payment gateway
- Secure online transactions

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/sreelekhanampally/artist-marketplace
```

---

### Backend

```bash
cd backend
npm install
npm start
```

Runs on

```
http://localhost:5000
```

---

### Frontend

```bash
cd frontend/artistic-portfolio
npm install
npm start
```

Runs on

```
http://localhost:3000
```
